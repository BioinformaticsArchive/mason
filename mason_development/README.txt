
"mason_core" contains the Javascript for core Mason.
	it contains a number of Javascript files that are combined 
	to make the final "mason_viewer.js" file for distribution.
	"mason_viewer-min-munged.js" is a minified version of "mason_viewer.js"

"mason_registry" contains the Javascript for the Mason Registry,
	code that makes connections between Mason viewers for 
	the vertical alignment lines.
